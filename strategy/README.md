## 推荐策略组合

### 策略A：精准狙击（高胜率低频率）

**入场条件**（同时满足）：

| 条件 | 规则 |
|------|------|
| 1D SRSI | 做多：<20 | 做空：>80 |
| 1D CCI | 做多：<-100 | 做空：>100 |
| 1D DMI | 做多：多 | 做空：空 |
| 1D ADX | >20（有趋势） |

**出场**：
- 止盈：+5% 或 24H后平仓
- 止损：-3%

**预期**：胜率60%+，信号稀疏（日均2-3个）

### 策略B：趋势跟踪（中频稳健）

**入场条件**：

| 条件 | 规则 |
|------|------|
| 布林带 | ≥2个时间框架突破同向 |
| 1D DMI | 与布林方向一致 |
| SRSI | 不在极端反向区域 |

**出场**：%B回到0.3-0.7区间 或 48H

**预期**：胜率47%+，信号充足

### 策略C：EMA趋势（高频基础）

**入场条件**：

| 条件 | 规则 |
|------|------|
| EMA(12,26) | ≥2TF快线>慢线=多，反之为空 |
| 1H/4H ADX | >20（避免横盘） |

**出场**：EMA交叉反转 或 24H

**预期**：胜率44%，信号极多

## 关键结论

1. **1D决定性**：1D方向同向确认后胜率提升0.3%
2. **空头优于多头**：1D=空胜率42%，1D=多仅38%
3. **CCI是最好的确认指标**：SRSI+CCI组合胜率飙升到61.7%
4. **强趋势(ADX>25)胜率41%** > 弱趋势(ADX<20)胜率34%
5. **高分≠高胜率**：≥8分信号仅3个且全败
